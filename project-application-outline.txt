项目申报书

1. 项目名称
- MoonTemplate 字符串模板渲染库

2. 项目简介
- MoonTemplate 是基于 MoonBit 的通用文本模板引擎，面向开发者工具与自动化生成场景。
- 提供变量插值、默认值、转义、条件渲染与循环渲染等核心能力，用于高效生成邮件、文档、日志、配置片段及 CLI 输出模板。

3. 项目方向与适用场景
- 方向：开发者工具 / 通用文本模板引擎 / 自动化生成
- 典型场景：
  - 邮件、通知、日志、配置片段的模板化生成
  - 文档/报告批量渲染
  - CLI 工具与脚手架的输出模板

4. 拟实现的核心功能
- 变量插值：支持 {{name}} 从数据上下文取值并替换
- 默认值：支持 {{name | default("unknown")}}，避免缺失字段导致输出异常
- 基本转义：默认转义特殊字符，避免格式破坏或模板注入
- 条件渲染：支持 {{#if cond}}...{{/if}} 实现条件渲染
- 循环渲染：支持 {{#each list}}...{{/each}} 进行列表渲染
